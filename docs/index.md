# 《机器人系统》课程

成都信息工程大学 软件工程学院

## 课程简介

本课程涵盖机器人系统的基本组成、传感器与执行器、运动学与动力学、控制方法、智能算法及实际应用案例。

## 课程章节概览

| 章号 | 章节标题 | 内容简介 |
|------|----------|----------|
| 第1章 | 机器人基础 | 涵盖机器人概论（定义、历史、结构、分类）、嵌入式系统基础（体系结构、STM32芯片）、最小系统与开发板介绍，为后续学习奠定基础。 |
| 第2章 | STM32CubeMX编程指南 | 介绍STM32CubeMX的安装与界面、创建工程的完整流程、GPIO配置实战（LED闪烁、按键检测）以及常用外设配置（USART、TIM、I2C、SPI、ADC）。 |
| 第3章 | 基于PicSimlab仿真器与CubeMX的典型软件开发 | 以PicSimlab仿真环境为舞台，结合STM32CubeMX与STM32CubeIDE，通过五个典型项目（GPIO流水灯、PWM呼吸灯、ADC数字电压表、USART通信控制台、智能温控风扇）系统覆盖STM32最常用的基础接口编程。 |
| 第4章 | 嵌入式软件的设计模式 | 讲解嵌入式软件设计模式，包括分层架构（HAL硬件抽象层、驱动层与应用层解耦）、状态机模式（有限状态机原理、Switch-Case实现、表驱动实现、层次化状态机），提升代码质量与可维护性。 |
| 第5章 | 嵌入式基于Renode仿真器的高级仿真编程 | 介绍Renode仿真器的体系结构与高级特性，深入讲解面向研究生层次的高保真嵌入式仿真方法、可重复性测试与自动化验证流程，并通过工程实例说明如何用Renode建立端到端仿真环境。 |
| 第6章 | 嵌入式实时操作系统 — FreeRTOS | 讲解嵌入式实时操作系统（RTOS）概念、FreeRTOS内核架构与组件、任务调度原理、进程间通信与同步、内存管理策略、中断与ISR安全API、移植与配置、调试与性能分析，以及基于FreeRTOS的工程实例。 |
| 第7章 | STM32定时器的应用 | 介绍STM32定时器的应用，包括定时器基础概念（时钟来源、预分频器、自动重装载寄存器、捕获/比较寄存器）、计数模式（向上计数、向下计数、中心对齐模式）、输出比较与PWM生成。 |
| 第8章 | PWM波与电机驱动 | 讲解PWM（脉宽调制）波形的等效原理、产生方法，以及在电机驱动中的应用，包括行星电机驱动原理、PWM占空比控制、电机速度调节等内容。 |
| 第9章 | 闭环控制与PID | 讲解闭环反馈控制基本理论、PID控制器原理与分析方法、离散化与嵌入式实现要点、控制器调参方法（Ziegler–Nichols、频域整定），以及工程实例（直流电机速度闭环控制）。 |
| 第10章 | 嵌入式系统与互联网 | 基于ESP8266与ESP32的无线连接（WiFi、蓝牙BLE/BT）实现方法、嵌入式设备如何安全接入互联网、常见NAT穿越策略（用于远程控制）、以及MQTT在物联网中的工程实践与安全应用。 |
| 第11章 | 课程工程实践项目 | 提供工业现场数据采集与监控系统等综合工程实践项目，涵盖传感器层、嵌入式主控层、通信层、上位机监控层的完整系统设计与实现。 |
| 第12章 | 嵌入式开发环境配置 | 系统介绍CubeIDE与Cube系列（含GitHub Copilot插件）、VSCode开发环境（含GitHub Copilot、CLI、CubeMX插件、基于容器的嵌入式编译环境）、Git在嵌入式开发中的使用等内容。 |
